Jenka Coffee Database

init_production.sql is the legacy PostgreSQL bootstrap file.
For an existing VPS database, run migration/normalize_snake_case_schema.sql after pulling source.

Use it for a fresh deploy:

psql -U <db_user> -d <db_name> -f src/main/resources/db/init_production.sql

Notes:
- The script drops and recreates the application tables.
- Booking and voucher tables/columns are intentionally not part of the schema.
- Numeric order IDs remain internal; customer-facing order URLs use orderCode.
- Physical PostgreSQL tables/columns should use snake_case, for example order_code and product_images.
